Introdução¶

Ja dizia o velho ditado, ‚ÄúCut Your Losses and Let Your Profits Run‚Äù. Aqui estamos interessados em cortar nossas perdas atrav√©s de um Stop Loss baseando-se no quantil de um movimento Browniano geom√©trico. Construção¶

Um processo estoc√°stico $\{S_t\}_{t \geq 0}$ pode ser modelado como um mBg (movimento Browniano geom√©trico) se satisfazer a SDE:

$$dS_t = \mu S_t dt + \sigma S_t dB_t$$

Neste caso, a solu√ß√£o da SDE acima √© dada por $S_t = S_0 e^{X(t)}$, onde $X(t) = \left(\mu - \frac{\sigma^2}{2} \right)t + \sigma B_t$ e $B_t$ √© um movimento Browniano, ou seja, $X(t)$ √© um movimento Browniano com drift $\left(\mu - \frac{\sigma^2}{2}\right)$.

Propriedades¶

$\quad$ i. $\lim_{t \to +\infty} \mathbb{P}\left(S_t = 0 \mid \mu - \frac{\sigma^2}{2} < 0\right) = 1$

[Demo] Dado que $\mu - \frac{\sigma^2}{2} < 0$ podemos concluir diretamente o resultado: $$\lim_{t \to +\infty} = S_0 e^{\left(\mu - \frac{\sigma^2}{2} \right)t + \sigma B_t} = 0$$

$\quad$ ii. $\lim_{t \to +\infty} \mathbb{P}\left(S_t = +\infty \mid \mu - \frac{\sigma^2}{2} > 0\right) = 1$. (Demonstra√ß√£o id√™ntica ao da propriedade i.)

iii. caso $\mu = \frac{\sigma^2}{2}$ temos que o limite $\lim_{t \to +\infty} = S_0 e^{\left(\mu - \frac{\sigma^2}{2} \right)t + \sigma B_t}$ √© indefinido pois $\sigma B_t \sim \mathcal{N}(0, \sigma^2 t)$.

CUIDADOS - Talvez seja interessante analisar a velocidade de converg√™ncia de (i) e (ii). Suponha que voc√™ ir√° abrir uma posi√ß√£o de long em um ativo, seria interessante escolher aquele que diverge o mais r√°pido poss√≠vel para infinito.

Distribuição Lognormal¶

Diremos que $\frac{S_t}{S_0}$ distribui lognormal se $ln \frac{S_t}{S_0} \sim N(\mu, \sigma^2)$. Neste caso, seu quantil √© definido como:

$$q(p) = exp(\mu + \sigma \phi^{-1}(p)) \quad p \in (0, 1)$$

Perceba que $\mu + \sigma \phi^{-1}(p)$ √© a quantil de uma normal, isto acontece pois os quantis tem a propriedade de serem preservados sob transforma√ß√µes monot√¥nicas. Vale notar que $\phi^{-1}(p)$ √© a inversa da CDF (Cumulative Distribution Function) de uma Normal padr√£o i.e. de $\mathcal{N}(0,1)$.

Pela solu√ß√£o de uma mBg, sabemos que:

$$ln \frac{S_t}{S_0} = X(t) = \left(\mu - \frac{\sigma^2}{2} \right)t + \sigma B_t \sim \mathcal{N}\left( \left[\mu - \frac{\sigma^2}{2}\right]t, \sigma^2 t \right)$$

Como $ln \frac{S_t}{S_0}$ distribui lognormal, temos que seu quantil √© definido como: $$q(p) = exp\left(\left(\mu - \frac{\sigma^2}{2}\right)t + \sigma\sqrt{t} \phi^{-1}(p)\right) \quad p \in (0,1)$$

class gBm():
    """
        Geometric Brownian Motion model.
                dS_t = Œº S_t dt + œÉ S_t dB_t
        
        Crescimento Exponencial. 
        o modelo tem um fit melhor para growth stocks como AAPL, TSLA, MGLU3, PRIO3 etc i.e.
        para ativos que tem um Crescimento Exponencial
    """
    
    def __init__(self, S):
        self.S = S
        self.estimateParams()
    
    def estimateParams(self):
        """
        Refer√™ncia:
        "Estimation of Geometric Brownian Motion Parameters for Oil Price Analysis" C. Jakob et al.
        """
        S = self.S
        X = np.diff(np.log(S), n=1)
        m = X.mean() #mean
        œÉ = X.std() #standard deviation
        Œº = m + ((œÉ**2)/2) #drift
        n = len(S)

        self.m = m
        self.œÉ = œÉ
        self.Œº = Œº
        self.n = n
    
    def E(self, t):
        """
        Refer√™ncia:
        Ross, Sheldon M. (2014). "Variations on Brownian Motion".
        Introduction to Probability Models (11th ed.).
        """
        S = self.S
        S0 = S[0]
        Œº = self.Œº
        return S0*np.exp(Œº*t)
    
    def Var(self, t):
        """
        Refer√™ncia:
        Ross, Sheldon M. (2014). "Variations on Brownian Motion". 
        Introduction to Probability Models (11th ed.).
        """
        S = self.S
        S0 = S[0]
        Œº = self.Œº
        œÉ = self.œÉ
        return (S0**2)*np.exp(2*Œº*t)*(np.exp((œÉ**2)*t) - 1)
    
    def q(self, p, t):
        """
         quantil de St/S0 o qual √© definido como:
                q(p) = exp( (Œº - œÉ**2/2)*t + œÉ*np.sqrt(t)*inv_Œ¶(p))
         p ‚àà (0,1)
        """
        #assert p>0 and p<1
        #assert type(t)==int

        œÉ = self.œÉ
        Œº = self.Œº
        
        mean = (Œº - (œÉ**2/2))*t
        var = œÉ**2*t
        return np.exp(mean +  np.sqrt(var)*N.ppf(p, 0, 1))

Stop loss e take profit¶

Supondo que o processo continue seu comportamento para os param√™tros $\mu$ e $\sigma$ estimados, faremos com que a solu√ß√£o inicial para a SDE apresentada i.e. $S_0$ seja igual a $S_T$ onde $T$ √© o tempo do √∫ltimo pre√ßo dispon√≠vel. Portanto, ao fazer isto estamos supondo que o comportamento do sistema (Par√¢metros) se permanecer√° inalterado, em outras palavras, dados do passado refletem dados do futuro (O que sabemos muito bem que nem sempre √© verdade). Podemos a partir disto criar um stop loss e um take profit baseado no quantil. Por exemplo, $q(0.2)$ representaria os 20% piores cen√°rios que um processo estoc√°stico $S_t$ poderia tomar dado os par√¢metros $\mu$ e $\sigma$.
